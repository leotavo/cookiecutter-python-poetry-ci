# Security Policy

## Supported versions

The latest release is supported; older tags are provided as-is.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Instead, report privately via [GitHub Security Advisories](https://github.com/{{ cookiecutter.gh_owner }}/{{ cookiecutter.repo_name }}/security/advisories/new), or email the maintainer at `{{ cookiecutter.author_email }}`.

You can expect an acknowledgement within a few days. Once the issue is confirmed and fixed, a patched release will be published and the reporter credited (unless they prefer to remain anonymous).
