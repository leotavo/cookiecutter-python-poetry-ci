# Cookiecutter: Python + Poetry + CI

Gera um projeto Python com Poetry (layout src), CLI mínima, pre-commit (ruff/black), mypy/pytest e CI no GitHub Actions.

Uso rápido

1) Instale cookiecutter (sugestão: pipx)

```bash
pipx install cookiecutter
```

2) Gere um projeto a partir deste template

```bash
cookiecutter gh:leotavo/cookiecutter-python-poetry-ci
```

3) Entre no diretório gerado e siga o README.
